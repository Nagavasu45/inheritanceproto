Q1. Create 2 objects - parent and child. Attach methods to parent and use those methods in child object using parents prototype 

    let parent={
        calc:function(a,b){
            console.log(a-b);
            console.log(a+b);
        }
    }
    let child=Object.create(parent);
    console.log("first object")
    child.calc(10,5)

output: first object 5 15 second object 10 30

explaination: in this code i have created two objects like parent object and child object and then in the child create object refernce to the parent object and parent object has one method like calc with 2 aruments after that call the child.calc(10,5)then we get the ouput
